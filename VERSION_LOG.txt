v1.0.0-mac  (2026)                               [CURRENT]
  Port จาก Windows v2.1.0
  • เสียงสะกด: win32com SAPI → macOS built-in 'say' command
  • Launcher: .vbs/.bat → .command/.sh
  • detection 'say': ใช้ shutil.which (ไม่มีเสียง ไม่มี subprocess)
  • stop(): proc.kill() บน Popen object (ไม่ใช้ pkill)
  • pygame: pre_init(44100, -16, 2, 512) — Apple Silicon fix
  • Tkinter: bind <Option-x> คู่กับ <Alt-x> ทุกปุ่ม
  • Tkinter: bind <Meta-z/x> คู่กับ <Control-z/x> (Cmd key)
  • asyncio: WindowsSelectorEventLoopPolicy ใช้ hasattr() guard
  หมายเหตุ: UI changes จาก Windows v2.2.0 ยังไม่ได้ port
  File: beetrainer_mac.py
